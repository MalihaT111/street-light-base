Troubleshooting

If login doesn't redirect:

  1. Check browser console for errors (F12 → Console)
  2. Verify React Router is working
  3. Check that useNavigate is imported in Login.jsx
  4. Verify form validation passes (password must be 8+ characters)

If Home page shows "Welcome back, Citizen 👋":

  1. Check localStorage has user data
  2. Verify user object has username property
  3. Check JSON parsing in Home.jsx useEffect

Common Issues:

  • Form validation errors: Password must be at least 8 characters
  • Email format: Must be valid email format
  • React Router: Make sure BrowserRouter is set up in main.jsx
  • Console errors: Check for any JavaScript errors
